## Typography
```scss
$ff-sans: 'Open Sans', sans-serif;
$ff-header: $ff-sans;
$ff-fixed: 'Courier New', Courier, fixed;

$light: 300;
$regular: 400;
$bold: 700;
```

*Usage*

```scss
p.sans {
  font-family: $ff-sans;
  font-weight: $light;
}
```
